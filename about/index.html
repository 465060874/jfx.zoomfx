ZoomFX

Zoomable pane for JavaFX.

About ZoomFX

ZoomFX project aims to deliver a high-level JavaFX container with support for zooming its content in and out, mouse-panning (and other things typically found in graphics editing applications), that would be usable right away, without the need to deal with all the transformations and input handling required to produce the desired results.

ZoomFX container features:

  • Zoom factor can be set via property, or via predefined high-level methods (these can be directly bound to desired user input actions, like mouse scroll)
  • Contents can be panned via scrollbars, or via predefined high-level methods (these can be directly bound to desired user input actions, like mouse drag)
